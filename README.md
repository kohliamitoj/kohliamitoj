## 💫 **About Me:**

I'm Amitoj Singh Kohli, a data enthusiast passionate about leveraging technology to solve complex problems and drive insights from data. Here’s a bit about me:

As a GenAI Engineer at Writer Information, I work on building production grade AI systems that transform unstructured data into actionable insights. My work focuses on multimodal document intelligence, where I leverage Google Vertex AI, OCR, and Retrieval Augmented Generation (RAG) pipelines to extract, process, and structure complex data across industries.

I design and deploy scalable backend systems using FastAPI and asynchronous processing frameworks, enabling efficient data extraction, workflow orchestration, and real-time analytics.

With a Master’s degree in Data Analytics Engineering from Northeastern University (USA), I bring a strong foundation in machine learning, NLP, and statistical analysis. I’m interested in solving real-world problems using a combination of data analytics and modern AI systems, bridging the gap between experimentation and production.
